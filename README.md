# ふぃろsophia

## サイト概要
アンケート投稿サイト

### サイトテーマ
論文を作る人のためのアンケートサイト！

### テーマを選んだ理由
「論文を書くために統計を取らないといけない！」でも、「アンケートの協力をしてくれる人が少ない！」と感じた経験はありませんか?
大学生の時にあったら便利と思ったアンケートを取ることに特化したサイトを作りました!

### ターゲットユーザ
論文を書く大学生や教授など

### 主な利用シーン
論文作成の際に必要なアンケート収集
